# Convention
- change: 기존에 있는 기능에서 변경사항이 생긴 경우
- feat: 새로운 기능 추가
- fix: 버그 픽스
- docs: 문서 수정
- resource: 리소스 추가(string, image 등)
- style: 세미콜론 수정, 코드 정리 등 스타일에 대한 수정
- refactor: 코드 리펙토링
- test: 테스트 코드 추가
- chore: 빌드 관련 변경 사항
